## Project setup

```
pnpm install
```

## Compile and run the project

```bash
# development
pnpm dev

# production mode
pnpm preview
```

## Run tests

```bash
# unit tests
pnpm run test

# e2e tests
pnpm run test:e2e

# test coverage
pnpm run test:cov
```
